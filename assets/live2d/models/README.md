# Live2D Models

Place Cubism `model3.json` model folders here.

For a quick test model, run this from the repository root:

```powershell
.\tools\download_live2d_haru.ps1
```

That script downloads the official Live2D Cubism Web Samples `Haru` model into:

```text
assets/live2d/models/Haru/
```

Model source:

```text
https://github.com/Live2D/CubismWebSamples/tree/develop/Samples/Resources/Haru
```

Check the upstream Live2D sample license before redistributing bundled model files.
